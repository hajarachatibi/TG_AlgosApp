Guide d'utilisation

 

 

Interface principale

 Une fois on ouvre l'application cette interface s'ouvre, il faut tout d'abord choisir le type du graphe et puis cliquer sur 'Lancer' pour lancer le programme. 

 

 

Une fois le bouton 'Lancer' est cliqué, une zone blanche (Pour le dessin du graphe) apparait comme suit:

 


-Pour ajouter un sommet cliquez sur un emplacement vide et saisissez un label pour ce sommet.

-Pour lier deux sommets par un arrêt/arc cliquez(sélectionnez) le premier sommet après cliquez(sélectionnez) le deuxième et saisissez le coût (dans le cas si le graphe est pondéré).

-Pour supprimer un arrêt/arc refaites les étapes de sa création.

-Pour supprimer un sommet, cliquez droit sur le sommet, et sélectionnez supprimer.

-Pour déplacer un sommet, cliquez deux fois sur le sommet, bougez votre sommet comme vous le voulez, encore un clic droit pour relâcher

-Pour modifier un sommet/arrêt/arc il faut le supprimer et le créer a nouveau.

-Pour avoir des informations sur le sommet un cilc droit sur le sommet et choisir 'informations'.


Application des algorithmes

Les algorithmes supportés par cette application sont:

  • BFS/DFS: pour le parcours des graphes.
  • Prim/ Kruskal: pour éliminer les cycles et générer un arbre couvrant minimal.
  • Wareshall: pour la fermeture transitive des graphes.
  • Dijkstra/BelmanFord pour trouver les plus courts chemins à partir d’un sommet.
  • Ford Fulkerson : pour trouver le flux max entre un sommet source et un sommet destinataire (les deux sont détectés automatiquement par l’application).
  • Welch & Powell : pour faire le coloriage.

Ces algorithmes sont appliquable dans les conditions suivantes:

• BFS et DFS : vous devez sélectionner un sommet de départ.
• Wareshall : vous devez créer un graphe non vide.
• Prim/ Kruskal: vous devez créer un graphe non vide.
• Dijkstra : vous devez créer un graphe pondéré positivement et sélectionné un
sommet de départ.
• BelmanFord : vous devez créer un graphe pondéré et sélectionner un sommet de
départ.
• FordFolkerson : vous devez créer un graphe pondéré, orienté et contient un sommet source(avec dégrée Entrante=0) et un sommet de destination (avec dégrée Sortante=0).

• Welch & Powell: vous devez créer un graphe non vide.

 

Ces algorithmes sont disponible dans le menu des algorithmes:



 Afin d'avoir plus de controle sur votre graphe, le menu qui suit vous permet de changer les couleurs des sommets, arrêtes, labels, le parcours BFS et DFS et de controler le temps d'animation des algorithmes.


 

Cette fenêtre vous affiche la trace de l'algorithme détaillée.

 

 

La fenêtre qui suit, vous donne des informations supplimentaires sur votre graphe, (nombre de sommets, nombre d'arêtes, matrice d'adjacence avec la possibilité d'entrer une autre matrice, et l'application va dessiner le graphe correspondant automatiquement)

 

 

Ce menu vous permet de:

  1. Exporter un PDF avec les résultats eu.
  2. Recomencer dèsle début.
  3. Accéder à ce guide d'utilisation.
